## 工程伦理与研究方法的“核心原则”一览表

| 类别         | 原则名称           | 关键词                     |
|--------------|--------------------|----------------------------|
| **工程伦理** | 以人为本           | 安全、福祉、尊重人权       |
|              | 安全至上           | 风险预防、冗余设计         |
|              | 诚实守信           | 不造假、不隐瞒             |
|              | 公正公平           | 利益平衡、无歧视           |
|              | 可持续发展         | 环保、资源节约             |
|              | 职业责任           | 胜任、举报、终身学习       |
| **研究方法** | 客观性             | 无偏见、实事求是           |
|              | 可验证性           | 可重复、可检验             |
|              | 系统性             | 流程完整、逻辑清晰         |
|              | 严谨性             | 方法科学、数据可靠         |
|              | 创新性             | 新知识、新贡献             |
|              | 学术诚信           | 不抄、不造、不篡           |
|              | 贡献透明           | 署名合理、责任明确         |
